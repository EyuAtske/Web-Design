Front End Development

What is Frontend Development?

Frontend development is the practice of building the user-facing parts of a website or web application. It uses HTML for structure, styling CSS for presentation, and JavaScript for interactivity.

Key Concepts

HTML
is the standard language used to create and structure the content of web pages using elements like headings, paragraphs, links, and images.
CSS
is the language used to control the visual presentation of HTML elements, including layout, colors, fonts, and spacing, to make web pages look attractive and consistent.
Java Script
is a programming language that adds interactivity, dynamic behavior, and functionality to web pages, allowing them to respond to user actions in real time.

A short quote about frontend work: Design is how it works as much as how it looks.

What are the Concepts

Each HTML document begins with a !DOCTYPE html declaration and is organized into two main parts:
  • Head: contains metadata, the page title, and links to styles or scripts.
  • Body: contains the actual visible content shown in the browser.
HTML focuses on structure and semantics, meaning it tells browsers what each part of the page represents, not how it looks. The style is handled by CSS, and interactivity by JavaScript. It’s the backbone of the web, allowing text, multimedia, and links to connect pages into the global network we use every day.
With CSS, you can define colors, fonts, margins, borders, spacing, animations, and responsive layouts that adapt to different screen sizes and devices.
Styles can be applied in three main ways:
  1. Inline CSS: written directly inside an HTML tag.
  2. Internal CSS: written within a style tag in the HTML document’s head.
  3. External CSS: stored in a separate .css file and linked with link rel="stylesheet".
The term “cascading” means that when multiple styles apply to the same element, CSS follows a specific priority order (inline → internal → external → browser defaults) and uses selectors and specificity rules to decide which style takes effect. In essence, CSS transforms plain HTML into visually appealing,
user-friendly, and professional-looking web designs.
While HTML defines the structure of a page and CSS defines its style, JavaScript makes websites responsive to user actions, such as clicks, typing, scrolling, or loading new content without refreshing the page.
JavaScript code can be added in three main ways:
  1. Inline JS: written directly inside an HTML element using the onclick, onchange, or other event attributes.
  2. Internal JS: written within a script tag in the HTML document.
  3. External JS: stored in a separate .js file and linked with script src="file.js" /script
JavaScript is an essential language for modern web development because it
transforms static web pages into interactive, dynamic experiences, enhancing usability and engagement.
